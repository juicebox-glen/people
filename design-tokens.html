Typography scale (:root)

Use these instead of one-off font-size / letter-spacing values for new UI. Mono chrome and slide titles are separate from body prose.

Token Role
--type-prose-fluid-fs (+ weight / lh / ls)Editorial intros — studio beliefs & process, home services intros, process cards
--type-prose-long-fs (+ lh / weight)Case study paragraphs (.case-study__prose)
--type-display-line-fs (+ lh / ls)Large list lines — home services columns
--type-overline-fs · --type-overline-lsUppercase overlines — case study meta & materials, footer block labels, credit
--type-supporting-fs · --type-supporting-lhMeta values, metrics table, “all projects” link
--chrome-mono-fs · --chrome-mono-lsDM Mono UI — nav, studio hero label, shared with .nav__link rules
--slide-panel-title-fs · --slide-panel-title-lsJournal / Studio / Case study overlay h2 bar + product card titles

Default body remains 1rem / 300 / 1.5 — --font-body (DM Sans). Headlines use --font-headline (Ufficio 500).
